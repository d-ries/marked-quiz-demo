# marked-quiz-demo
This is a demo of a markdown file hosted on github on https://github.com/d-ries/marked-quiz-demo

This file contains an interactive quiz / question:

~QUIZ
Q: What is 2 + 2?
A) 3
B) 4
C) 5
D) 6
ANSWER: B
~

~QUIZ
Q: This is a sample question where the right answer is A?
A) Naaaa naaa naaa
B) nanananana
C) hey
D) jude
ANSWER: A
~

This file has been parsed using the tool https://d-ries.github.io/markdown/

Enjoy!
